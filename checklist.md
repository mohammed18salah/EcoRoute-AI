# Pre-Submission Checklist

## Functional Testing
- [ ] **Geocoding**: Type "Paris" and "Berlin", ensure it doesn't crash.
- [ ] **Routing**: Click "Find Eco Routes", ensure lines appear on map.
- [ ] **API Key**: Verify ORS_API_KEY is in `.env.local` and loaded. 
      *(Note: If map lines are missing, check console logic for API limits).*

## Visual Polish
- [ ] **Responsiveness**: Resize browser to mobile width; ensure cards stack correctly.
- [ ] **Images**: Check header and background images load.

## Content
- [ ] **About Section**: Read through the "How it works" text in the bottom section.
- [ ] **Title**: Ensure "EcoRoute AI" is visible in the tab/header.

## Submission Materials
- [ ] Record 2-min demo video (use the script in README).
- [ ] Push code to GitHub.
- [ ] Paste Repository URL into Devpost.
